# Cat/Plant Peaceamker

This is a demo for the Hack 4 of CS 839 - Building Interactive System. This system uses ESP32 hardware to detect the cat and prevent the cat approaching the plant by buzzing. It aims to be a peacemaker for your cat and the plant when you are not present!

## Website Demo

This demo website can work independently for demo purpose. In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
